## Setup Instructions

### Option 1: Import via UI
1. Open LM Studio
2. Go to Settings > Presets
3. Click "Import Preset"
4. Select this JSON file

### Option 2: Manual Installation
```bash
# Copy to LM Studio presets directory
cp backend-kit.json ~/.lmstudio/presets/

# On Windows
copy backend-kit.json %USERPROFILE%\.lmstudio\presets\
```

### Option 3: Use in Chat
1. Start a new chat in LM Studio
2. Click the preset dropdown
3. Select "Backend Engineering Kit"
4. Start coding!

## Recommended Models
