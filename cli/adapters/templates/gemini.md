# Gemini CLI Instructions

> For Google Gemini CLI
> Use as context or system prompt
> Generated by Backend Engineering Kit (bek)

## Context

This project follows Backend Engineering Kit patterns for production backend code.

## Workflow

### Parse → Search → Apply → Validate

1. **Parse**: Identify scope (api, database, security, reliability, observability)
2. **Search**: `bek search "<query>" --scope <scope>`
3. **Apply**: Follow pattern's Solution section
4. **Validate**: `bek gate --checklist <id>`

## Code Generation Guidelines

### TypeScript/Node.js
- Use strict mode
- ESM modules
- Async/await
- Zod for validation

### Python
- Type hints required
- Pydantic models
- FastAPI/HTTPX

### Go
- Standard library preferred
- Context propagation
- Error wrapping
