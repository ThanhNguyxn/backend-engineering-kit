# Codex Instructions for Backend Kit

## Knowledge Base
This project has a Production Backend Kit with patterns for:
- API design
- Database operations
- Security controls
- Reliability patterns
- Observability

## CLI Commands
- `kit search "<query>"` - Find relevant patterns
- `kit gate --checklist <id>` - Run quality checklist
- `kit list --scope <scope>` - List patterns by scope

## Usage
1. Search for patterns related to the task
2. Apply the Solution from matched pattern
3. Validate with appropriate checklist
