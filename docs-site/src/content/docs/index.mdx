## 60-Second Quickstart

```bash
# Install globally
npm install -g production-backend-kit

# Verify installation
bek doctor

# Initialize a new project with a preset
bek init --preset node-express

# Run quality gate checklist
bek gate --checklist checklist-prod-readiness
```

## Available Presets

| Preset | Description |
|--------|-------------|
| `node-express` | Express.js API with error handling, validation, observability |
| `node-fastify` | Fastify with Pino, Zod, Docker, CI |
| `node-minimal` | Minimal TypeScript + ESM starter |
| `saas-multitenant-lite` | Multi-tenant SaaS patterns (RBAC, billing, data isolation) |
